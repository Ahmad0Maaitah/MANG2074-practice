# MANG2074 Financial Econometrics 1 — Practice Arcade

Interactive practice site for MANG2074 (University of Southampton, 2026-27, module leader Ahmad Maaitah).

**Start here → [index.html](index.html)** (or the GitHub Pages link once enabled).

Eleven self-contained pages, one per lecture topic (weeks 1-11). Each page has three parts:

1. **🔬 Simulation** — sliders and live charts to build intuition (fit your own OLS line, generate AR(1) paths, drive a GARCH volatility engine, switch market regimes...).
2. **🎮 Game** — a 10-round scored game per topic: *Normal or Not?*, *Guess the Beta*, *Significant or Spurious?*, *Regression Doctor*, *Match the Correlogram*, *Forecast Duel*, *Calm or Crisis?*, *Risk Manager*, *Regime Detective*, *Hausman Court*, *Research Sprint*. Best scores are saved in your browser.
3. **⌨️ Code challenges** — fill-the-blank questions with a Python/R toggle matching the module's two code repositories.

No installation needed: every page is plain HTML/JavaScript with no external dependencies — it works from a web link or by double-clicking the file. Best scores are stored only in the student's own browser (localStorage); no data is collected.

These materials accompany the MANG2074 teaching repositories and the textbook *Introductory Econometrics for Finance* (Brooks, 4th ed.). All games and simulations are original teaching material.
